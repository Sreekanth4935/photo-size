- **Resize by pixels**: Planned for future implementation.

- **Batch resize**: Planned for future implementation.
- **Crop and rotate**: Planned for future implementation.

- **Format preservation**: Planned for future implementation.

- **No Login Required**: App works fully offline
- **Local-Only Storage**: All data stays on-device
- **No Unnecessary Permissions**: Only request what is needed
- **Privacy Policy**: Will be hosted externally and linked in-app
